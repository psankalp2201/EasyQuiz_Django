About EasyQuiz

EasyQuiz is built to provide an easy and cost efficient platform for hosting and participating quizes.Designed with the help of one of the most powerful and polular backend framework "Django", it provides efficient perfomance. Designed with the aim of keeping things simple yet useful, EasyQuiz provides a very simple and minimal UI which is capable of doing all the assigned tasks. EasyQuiz is still in its development phase hence it has a long way to go but still much capable for hosting quizes at college/instituional level.
