Road Map to EasyQuiz

August 28th
On August 28th this project was started. We did a brief research on possible schema of the models.We also tried to identify the problems that we could face later while implementing the schema and thought of possible solutions.
August 30th
The development of Templates started and the aim was to create a very basic UI with the help of HTML,CSS,Bootstrap where we could implement our Django logic. Hence we create a decent looking Frontend in 2 days.
September 1st
We designed multiple models for multiple purposes in Django. We also designed a Signup and Login System for both admins and users and tested it.Thankfully both of the modules ran perfectly without any flaws.

September 2nd
On this day we started our main work that was to create admin panel which was quite challenging. But with the help of our Trainer we were able to implement one of the most challenging part of out project that was Add_Question Module. Then with next 1-2 days we were able to complete our Admin Panel.
September 4th
Along side our Admin panel where major work was already completed we started out work on user panel which was quiet easy since it was largely based on Add_Question Module. Hence we completed it on same day.
September 5th
Now we have almost completed 90% of our project. One this day we implemented Session Tracking on our website and also started to complete all the missing things and groomed the Frontend a bit. And Completed our Project on 7th of September.
